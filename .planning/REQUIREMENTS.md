## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Live chat support | Dutch market prefers email/phone, requires staffing |
| Virtual room visualizer | High complexity, low conversion impact for blinds |
| Per-city landing pages | Thin content penalty risk, maintenance burden |
| Multi-language support | Netherlands-only market sufficient, avoid splitting focus |
| Product reviews on pages | New shop has no reviews yet, empty sections hurt trust |
| Real-time price comparison widgets | Drives customers away before value established |
| Video content (installation guides) | Text + images sufficient, video production expensive |
| Contact form backend | Frontend form only, backend submission deferred to separate effort |
| i18n routing framework | Single Dutch locale, inline content sufficient |
